# Changelog

All notable changes to NGC Alliance Hub will be documented in this file.

## [1.0.0] - 2025-01-15

### Added
- **Alliance Hub (Index)**: Main dashboard with multi-language support (IT/EN/FR)
- **War Room Defense Tracker**: Real-time power tracking with 3D wheel interface
- **Strategy Manager**: Interactive Canyon & Desert Storm coordination tool
  - Building assignments with drag-and-drop
  - Team A/B management
  - Kill Swat and Reserves tracking
  - Formation history and load system
  - **NEW:** "Load Last Battle" button to restore complete formations
- **Personal Stats Dashboard**: Growth analytics with dual chart system
  - Total Power chart (large, prominent)
  - Squad Details chart (Missiles, Aircraft, Tanks)
- **Google Apps Script Backend**: Self-hosted database system
  - Zero infrastructure costs
  - Complete data sovereignty
  - Automatic synchronization
